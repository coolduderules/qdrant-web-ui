## Filter - Match

``` json withRunButton="true"
POST collections/terraforming/points/scroll
{
  "filter": {
    "must": [
      {
          "key": "color",
          "match": {
          "value": "black"
          }
      }
    ]
  },
  "limit": 3,
  "with_payload": true
}
```
