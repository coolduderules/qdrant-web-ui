## Filter 

``` json withRunButton="true"
POST collections/computer_store/points/scroll
{
  "filter": {
    "must": [
      {
       "key": "price",
       "range": {
        "gt": null,
        "gte": null,
        "lt": 1000,
        "lte": null
        }
      }
    ]
  },
  "limit": 3,
  "with_payload": true
}
```
